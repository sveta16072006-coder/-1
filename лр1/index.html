История создания языка C#

Язык программирования C# (произносится как "си шарп") был разработан компанией Microsoft в период с 1999 по 2002 год под руководством Андерса Хейлсберга и Скотта Вильтаумота. Основная цель создания языка — предоставить современный, простой в использовании, объектно-ориентированный язык для платформы .NET.

Название "C#" происходит от музыкального знака диез (♯), который указывает на повышение ноты на полтона. Это символизирует эволюцию языка C++ — "на полтона выше C++".

Основные вехи развития C#:

  1. Июнь 2000 — Первое публичное представление языка на конференции Professional Developers Conference
  2. Январь 2002 — Выпуск первой версии C# 1.0 вместе с Visual Studio .NET
  3. Ноябрь 2005 — C# 2.0 с добавлением обобщений (generics)
  4. Ноябрь 2007 — C# 3.0 с LINQ и лямбда-выражениями
  5. Апрель 2010 — C# 4.0 с динамической типизацией
  6. Август 2012 — C# 5.0 с async/await
  7. Июль 2015 — C# 6.0 с упрощенным синтаксисом
  8. Март 2017 — C# 7.0 с кортежами и сопоставлением с образцом
  9. Сентябрь 2019 — C# 8.0 с nullable reference types
  10. Ноябрь 2020 — C# 9.0 с записями (records)
  11. Ноябрь 2021 — C# 10.0 с улучшениями для минимальных API
  12. Ноябрь 2022 — C# 11.0 с raw string literals

Краткий обзор языка C#

C# — это современный, объектно-ориентированный, типобезопасный язык программирования, который позволяет разработчикам создавать различные типы безопасных и надежных приложений, работающих на платформе .NET.

Ключевые особенности C#

  • Объектно-ориентированность — полная поддержка ООП: инкапсуляция, наследование, полиморфизм
  • Компонентно-ориентированный — свойства, события, атрибуты
  • Типобезопасность — строгая проверка типов во время компиляции
  • Автоматическое управление памятью — сборка мусора (garbage collection)
  • Универсальные типы (generics) — повторно используемый типобезопасный код
  • LINQ (Language Integrated Query) — встроенный язык запросов
  • Асинхронное программирование — ключевые слова async/await
  • Кроссплатформенность — благодаря .NET Core и .NET 5+

Области применения C#

  • Веб-приложения (ASP.NET Core)
  • Десктопные приложения (Windows Forms, WPF)
  • Мобильные приложения (Xamarin)
  • Игры (Unity game engine)
  • Микросервисы и облачные приложения
  • Машинное обучение (ML.NET)
  • Интернет вещей (IoT)

Примеры кода на C#

Ниже представлены различные примеры кода на C# с подсветкой синтаксиса, демонстрирующие основные возможности языка.

1. Простая программа "Hello, World!"

Традиционная первая программа на любом языке программирования:

using System;

namespace HelloWorld
{
    class Program
    {
        static void Main(string[] args)
        {
            // Вывод текста в консоль
            Console.WriteLine("Hello, World!");
            
            // Ожидание нажатия клавиши
            Console.WriteLine("Нажмите любую клавишу для выхода...");
            Console.ReadKey();
        }
    }
}

Этот код создает простую консольную программу, которая выводит текст в консоль.

2. Классы и объекты в C#

Пример определения класса, создания объектов и работы с ними:

using System;

namespace OOPExample
{
    // Базовый класс
    public class Person
    {
        // Поля (fields)
        private string name;
        private int age;
        
        // Свойства (properties)
        public string Name 
        { 
            get { return name; }
            set { name = value; }
        }
        
        public int Age
        {
            get { return age; }
            set 
            { 
                if (value >= 0) 
                    age = value; 
            }
        }
        
        // Конструктор
        public Person(string name, int age)
        {
            this.name = name;
            this.age = age;
        }
        
        // Метод
        public virtual void DisplayInfo()
        {
            Console.WriteLine($"Имя: {Name}, Возраст: {Age}");
        }
    }
    
    // Производный класс
    public class Employee : Person
    {
        public string Position { get; set; }
        
        public Employee(string name, int age, string position) 
            : base(name, age)
        {
            Position = position;
        }
        
        // Переопределение метода
        public override void DisplayInfo()
        {
            base.DisplayInfo();
            Console.WriteLine($"Должность: {Position}");
        }
    }
    
    class Program
    {
        static void Main()
        {
            // Создание объектов
            Person person = new Person("Иван", 30);
            Employee employee = new Employee("Анна", 25, "Разработчик");
            
            // Вызов методов
            person.DisplayInfo();
            employee.DisplayInfo();
        }
    }
}

3. Асинхронное программирование

Пример использования async/await для асинхронных операций:

using System;
using System.Net.Http;
using System.Threading.Tasks;

namespace AsyncExample
{
    class Program
    {
        static async Task Main(string[] args)
        {
            Console.WriteLine("Начало загрузки данных...");
            
            try
            {
                string data = await DownloadDataAsync("https://api.example.com/data");
                Console.WriteLine($"Данные загружены. Длина: {data.Length} символов");
                Console.WriteLine($"Первые 100 символов: {data.Substring(0, Math.Min(100, data.Length))}...");
            }
            catch (HttpRequestException ex)
            {
                Console.WriteLine($"Ошибка при загрузке данных: {ex.Message}");
            }
            
            Console.WriteLine("Программа завершена.");
        }
        
        static async Task DownloadDataAsync(string url)
        {
            using (HttpClient client = new HttpClient())
            {
                // Установка таймаута
                client.Timeout = TimeSpan.FromSeconds(30);
                
                // Асинхронная загрузка данных
                HttpResponseMessage response = await client.GetAsync(url);
                response.EnsureSuccessStatusCode();
                
                return await response.Content.ReadAsStringAsync();
            }
        }
    }
}

Пояснение к примеру:

  • async — указывает, что метод является асинхронным
  • await — приостанавливает выполнение метода до завершения асинхронной операции
  • Task и Task<T> — представляют асинхронные операции
  • Асинхронные методы не блокируют основной поток выполнения
